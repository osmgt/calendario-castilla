ℹ️ Información del Proyecto

🎯 Objetivo

Primer calendario automático del Real Madrid Castilla con horarios específicos para Guatemala, actualizado en tiempo real.

📡 Fuentes de Datos

FotMob API y datos oficiales del Real Madrid. Sistema de fallback inteligente.

🔄 Actualización

Cada 30 minutos automáticamente, con actualizaciones extra los días de partido.

🕐 Zona Horaria

America/Guatemala (GMT-6). Todos los horarios están ajustados automáticamente.

📱 Compatibilidad

iOS, Android, Google Calendar, Outlook, y cualquier aplicación que soporte calendarios ICS estándar.

🆓 Gratuito

Completamente gratis, sin límites de uso, creado por y para la comunidad madridista guatemalteca.
